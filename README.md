# blockchainmodule

## Launch tests
Conditions:
- must be in the Quest folder
- solution must be name [exercise]_sl.js
then:
`node test.mjs [exercise]`

Example:

```bash
~/blockchainmodule/$ cd Quest1\ Crypto/
~/blockchainmodule/Quest1 Crypto $ node test.mjs basicWallet
```
